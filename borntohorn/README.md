# Born To Horn

Static web app for browsing major-scale notes and standard fingerings across a practical written study range for:

- Bb trumpet
- French horn in F

## What it does

- Lets you toggle between trumpet and horn
- Filters the key menu by category such as all keys, jazz keys, blues keys, and brass-friendly keys
- Shows the selected concert key and the corresponding written key for the current instrument
- Lists every scale note from written C4 to written C6 with note names, concert pitch equivalents, and standard fingerings

## Run locally

Because this is a plain static site, you can open `index.html` directly in a browser or upload the folder to nearly any hosting provider.
